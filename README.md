# Balloon Adventure

A simple hot air balloon game built with PixiJS. Try to fly as high as you can and land safely before the balloon pops ðŸŽ‰

## Technologies

- PixiJS v8 - 2D rendering engine
- @pixi/sound - audio library for background music and sound effects
- Vite - development server and build tool
- JavaScript ES6+ modules

## How to run

```bash
npm install
npm run dev
```

THen open your browser and start playing. Click on the screen to turn on the sound effects, then click "Land Now" before your balloon pops to win!
