# Balloon Adventure

A simple hot air balloon game built with PixiJS. Try to fly as high as you can and land safely before the balloon pops

## Technologies

PixiJS v8 - 2D rendering engine
Vite - development server and build tool
JavaScript ES6+ modules

## How to run

```bash
npm install
npm run dev
```

THen open your browser and start playing. Click "Land Now" before your balloon pops to win!
